Sistema modal

Command mode

El modo por defecto. Con [ESC] volvemos a este modo.

Sub mode: Visual mode

Desde el command mode, con la tecla 'v' accedemos.

Insert mode

  • i: insertar antes del cursor
  • I: insertar antes del primer caracter que no sea de tipo espacio
  • a: insertar tras del cursor
  • A: insertar al final del a linea
  • o: insertar linea vacia bajo el cursor
  • O: insertar linea vacia sobre el cursor

Movimiento

  • h: izquierda
  • j: abajo
  • k: arriba
  • l: derecha
  • gg: inicio del archivo
  • G: fin del archivo
  • $: fin de linea
  • 0: inicio de linea
  • f{caracter}: desplazarse hasta {caracter}
  • F{caracter}: retroceder hasta {caracter}
  • w: siguiente inicio de palabra
  • b: anterior inicio de palabra
  • e: siguiente fin de palabra
  • ge: anterior fin de palabra

Acciones basicas

  • y: yank (copiar)
  • d: delete (cortar)
  • p: paste
  • u: undo

Algunos comandos

  • :! Ejecutar comando en el shell
  • :e /path para abrir otro archivo
  • :r para leer otro archivo
  • :r! para leer el resultado de un comando

Busquedas

  • /[REGEX]: buscar hacia abajo
  • ?[REGEX]: buscar hacia arriba
  • n: siguiente resultado
  • N: resultado anterior

Marcas

  • m{a-z}: crear marca en posicion del cursor
  • '{a-z}: ir a la marca
  • :marks: mostrar marcas
  • .: marca el ultimo sitio modificado
  • ": marca el sitio donde se salio del fichero la ultima vez
  • '': vuelve al sitio donde estabas antes de ir a una marca

Tabs

  • :tabnew: crea una nueva tab
  • :tabn: ves a la siguiente tab
  • :tabN: ves a la tab anterior

El idioma de VI

Lo basico

  • [numero][movimiento]
  • Combinar una accion con un movimiento
  • [accion][numero][movimiento]
  • Combinar una accion con una marca
  • ':': para escribir "frases" mas completas
  • '.': repetir frase anterior

Rangos

  • :[linea],[linea]
  • :[marca],[marca]
  • ':%': todo el archivo
  • ':.,$': desde el cursor hasta el final del archivo

Substituciones

  • :[]s/[Regex busqueda]/[regex reemplazo]/[opciones]
  • :%s/prueba/test/g
  • :%s/prueba/test/[g][e][i][c]
  • :.,$s/prueba/test/g
  • :'a,'bs/prueba/test/g

Opciones de las substituciones

  • c: pide confirmacion
  • g: reemplaza todas las ocurrencias
  • i: ignora capitalizacion
  • e: no muestres error si no encuentra resultados
